A professional, Blender-inspired web application designed to transform images and videos into high-fidelity ASCII art.

---

### 🚀 Key Features

| 🛠️ Processing | 🎮 Viewport | 📤 Export |
| :--- | :--- | :--- |
| **Real-time Engine**: Instant conversion of images and MP4 videos. | **Blender Navigation**: Pan with MMB, zoom with Scroll/Buttons. | **Multi-Format**: Export as PNG, TXT, or GitHub Markdown. |
| **Custom Ramps**: Define your own character gradients (Dark → Light). | **Precise Zoom**: Fine-grained 2% increments for perfect alignment. | **Embed Ready**: One-click copy for READMEs and documentation. |
| **Resolution Scale**: Adjust density dynamically for different displays. | **Centered Origin**: Smooth, jump-free panning system. | **Dark Studio UI**: Distraction-free creative environment. |

### 🛠️ Installation & Setup

Get started with Ascii-Studio in seconds:

#### 1. Clone the Repository
```bash
git clone https://github.com/tarunkumar-sys/art_to_ascii.git
cd art_to_ascii/ascii-app
```

#### 2. Install Dependencies
```bash
npm install
```

#### 3. Run Development Server
```bash
npm run dev
```
*The app will be available at `http://localhost:5173`*

---

### 📖 How to Use

1. **Upload**: Drag & drop or use the **Open Media File** button in the sidebar.
2. **Adjust**: Tweak the **Color Profile** and **Resolution Scale** to find the best look.
3. **Navigate**: Use the **Middle Mouse Button** to pan and **Scroll Wheel** to zoom (just like Blender).
4. **Export**: Choose your preferred format from the **Output & Export** panel to save your art.
